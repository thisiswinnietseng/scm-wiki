實體 SIM 卡 POS 出貨

實體出貨
⚠️ 作業時間：每天 11:00 前
  1. 1
    撈單

    BE2 → 訂單管理 → 訂單報表匯出；條件：郵寄 / 台灣郵寄共表 / KKday / 狀態：處理中＋已處理＋旅客資料已齊

    特殊：24324 香港卓一電訊、25356 PhoneBox 用供應商模式撈

  2. 2
    整理表單

    貼至「撈單」，電話欄 Z→A 排序分台灣 / 海外；備註特殊處理：

    • 指定時間寄送 → 通知客服
    • 管理室代收 → 補充地址
    • SIM 卡相關 → 詢問 BD

    海外訂單貼海外辦公室郵寄表並更新狀態；台灣訂單確認中文姓名（英文 → 退單）

  3. 3
    印標籤

    開「台灣郵寄共表」→ 🏷🔖📑 → 下載 xlsx → 開印標機系統 → 新連線 → 選 xlsx → 列印標籤

  4. 4
    揀貨

    開「當日應出貨品項數量」→ 前往 SIM 卡櫃 → 開 POS 系統選「總公司 OP 庫存」→ 管理訂單出貨 → 查詢訂單 → 核對品項 → 出貨 → 包裝 → 製作大宗普掛表（20 個要連號）→ 列印普掛表（兩份：給郵局 / 留底）

POS 系統操作說明

  • 控量商品：按數量後按出貨
  • 控貨商品：掃描條碼後按出貨
  • 顯示「已出貨成功」＝ 完成
⚠️ 郵寄需中文姓名；外國旅客可英文；非外國旅客留英文 → 退單詢問
